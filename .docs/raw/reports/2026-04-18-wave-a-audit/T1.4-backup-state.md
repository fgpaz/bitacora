## Backup strategy lockeada (brainstorming)

- Docker volume snapshot (tar -czf del mountpoint via container busybox, con stop/start del container PG para consistency)
- Retention: 30 dias local (`/var/backups/zitadel/`) + 90 dias offsite
- Cron: 03:00 UTC daily (00:00 ART, low-traffic)
- Primer snapshot post-config esperable ~50-200MB por ahora
- Downtime esperado: 30-90 segundos por snapshot (aceptable a esa hora)
