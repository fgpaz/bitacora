# Task T3.1: Backup docker volume snapshot + cron + primer backup offsite

## Shared Context
**Goal:** automatizar snapshot diario del volumen Docker de la Postgres Zitadel, con upload a destino offsite, retention 30d, y correr al menos 1 backup verificado.
**Stack:** bash + docker + rclone (o aws-cli) para offsite, cron en VPS turismo.
**Architecture:** backup strategy lockeada en brainstorming: docker volume snapshot + offsite. No pg_dump (user eligio C).

## Locked Decisions
- Snapshot via `docker run --rm -v <vol>:/data busybox tar -czf /backup/<ts>.tar.gz -C / data` (crea tar comprimido). Para Postgres, hay que pause writes con `docker exec <pgcontainer> pg_ctl stop` antes del tar **o** usar `pg_basebackup` dentro del container — recomendacion final depende de T1.4 findings. Fallback seguro: stop container antes de tar (1-2min downtime aceptable diario en madrugada).
- Destino offsite: por definir con user en este task (candidates de T1.4: B2, R2, Hetzner Storage Box, rsync desktop via Tailscale).
- Retention: 30 dias local + 90 dias offsite.
- Cron hora: 03:00 UTC (00:00 ART) — horario low-traffic.
- Script vivira en VPS en `/opt/zitadel-backup/snapshot.sh` + `/etc/cron.d/zitadel-backup`.
- Git-tracked local en `infra/backups/zitadel/snapshot.sh` + `infra/backups/zitadel/README.md`.

2. Escribir `infra/backups/zitadel/snapshot.sh` con el siguiente template (ajustar volumen y container del T1.4):
   ```bash
   #!/usr/bin/env bash
   set -euo pipefail

   # snapshot.sh — Zitadel Postgres volume backup
   # Requires: docker, rclone, env vars from /opt/zitadel-backup/.env

   source /opt/zitadel-backup/.env

   TS=$(date -u +%Y%m%d-%H%M%S)
   LOCAL_DIR=/var/backups/zitadel
   OFFSITE_REMOTE="${ZITADEL_BACKUP_OFFSITE_PROVIDER}:${ZITADEL_BACKUP_OFFSITE_BUCKET}"
   VOLUME="${ZITADEL_POSTGRES_VOLUME_NAME}"
   PG_CONTAINER="${ZITADEL_POSTGRES_CONTAINER_NAME}"
   RETENTION_LOCAL=30
   RETENTION_OFFSITE=90

   mkdir -p "$LOCAL_DIR"

   echo "[$(date -u -Iseconds)] Stopping Postgres container $PG_CONTAINER"
   docker stop "$PG_CONTAINER"

   echo "[$(date -u -Iseconds)] Creating snapshot"
   docker run --rm -v "${VOLUME}:/data:ro" -v "${LOCAL_DIR}:/backup" busybox \
     tar -czf "/backup/zitadel-pg-${TS}.tar.gz" -C / data

   echo "[$(date -u -Iseconds)] Starting Postgres container"
   docker start "$PG_CONTAINER"

   echo "[$(date -u -Iseconds)] Uploading to offsite $OFFSITE_REMOTE"
   rclone copy "$LOCAL_DIR/zitadel-pg-${TS}.tar.gz" "$OFFSITE_REMOTE/" --progress

   echo "[$(date -u -Iseconds)] Cleaning local backups older than ${RETENTION_LOCAL}d"
   find "$LOCAL_DIR" -name 'zitadel-pg-*.tar.gz' -mtime +$RETENTION_LOCAL -delete

   echo "[$(date -u -Iseconds)] Cleaning offsite backups older than ${RETENTION_OFFSITE}d"
   rclone delete --min-age "${RETENTION_OFFSITE}d" "$OFFSITE_REMOTE/"

   echo "[$(date -u -Iseconds)] Done: zitadel-pg-${TS}.tar.gz"
   ```
