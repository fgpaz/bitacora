## Inventario de Flujos

## Lectura operativa del canon

- `Implementado backend`: el runtime actual ya materializa la parte backend del flujo.
- `Parcial backend`: existe soporte backend parcial, pero el slice completo o su canal visible sigue pendiente.
- `Diferido`: el flujo sigue siendo canon vigente, pero no existe runtime que lo materialice hoy.

Los `FL-*` detallados siguen describiendo el comportamiento objetivo del MVP. Su estado actual debe leerse desde este índice antes de asumir que el flujo ya existe en código.
